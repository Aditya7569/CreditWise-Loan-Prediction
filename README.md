# CreditWise-Loan-Prediction
## Overview
CreditWise is a machine learning-based credit risk assessment and loan approval prediction system developed using Python and Scikit-learn.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Feature Engineering
- Loan Approval Prediction
- Model Comparison

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Naive Bayes

## Evaluation Metrics
- Precision
- Recall
- F1 Score

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Results
Compared multiple classification algorithms and selected the best-performing model based on evaluation metrics.
